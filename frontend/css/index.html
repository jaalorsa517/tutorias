Bienvenidos

Hoy aprenderemos de CSS y la metodología BEM

La idea de esta charla es mostrar la importancia de conocer el concepto de Especificidad en Css y de la manera correcta de usar la metodologia BEM

En ese orden de ideas, veremos

  • Especificidad

    Es el concepto donde se da prioridad a los selectores de CSS.

    Como se sabe, en css se puede tener selectores de Etiqueta, Atributo, Clases, Identificador. Adicionalmente, también se puede dar estilos en líneas y usar la palabra reservada !important.

    Todo lo anterior, se le puede dar un valor numérico, para ilustrar como el DOM le da prioridad al selector:

    Selector Valor numérico
    !important 10000
    Identificador 100
    Atributo 10
    Clase 10
    Etiqueta 1

  • BEM

    Es una metodología que nos permite tener un código más limpio y ordenado, además de que nos permite tener un código más escalable. También evita el problema común de sobrescritura de estilos, ya que este permite tener nombres únicos para las clases.

    La estructura de BEM es Bloque-Elemento-Modificador.
